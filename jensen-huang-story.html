💻 핵심 기술 이해하기

🎮 GPU (Graphics Processing Unit)

기존 그래픽 칩과의 차이:

• 이전: CPU를 보조하는 단순 장치
• GPU: 독립적으로 3D 그래픽을 처리하는 강력한 프로세서

핵심 기술: 병렬 처리

• CPU: 복잡한 작업을 순차적으로 처리 (한 번에 하나씩)
• GPU: 단순한 작업을 동시다발적으로 처리 (한 번에 수천 개)

비유:

• CPU = 천재 한 명이 문제를 순서대로 푸는 것
• GPU = 수천 명이 각자 간단한 문제를 동시에 푸는 것

⚡ CUDA (Compute Unified Device Architecture)

혁명적 전환점:

GPU를 그래픽뿐 아니라 범용 계산에도 사용할 수 있게 만든 소프트웨어 플랫폼

개발 시기: 2006년 (아무도 필요로 하지 않을 때)

투자 기간: 10년

결과: 2012년 딥러닝과 만나 빛을 발함

왜 중요한가?

• AI 학습은 단순 계산을 수백만 번 반복해야 함
• CPU로는 몇 달 걸릴 작업을 GPU+CUDA로 며칠 만에 완료
• 딥러닝 혁명의 핵심 인프라

🧠 딥러닝 (Deep Learning)

개념:

인공신경망이 사람의 뇌처럼 스스로 패턴을 배우고 학습하는 기술

왜 GPU가 필요한가?

• 수백만 개의 파라미터를 동시에 조정해야 함
• 방대한 데이터를 반복 처리해야 함
• GPU의 병렬 처리 능력이 완벽하게 적합

역사적 순간: AlexNet (2012년)

• 엔비디아 GPU 2개 + CUDA 사용
• 이미지 인식 대회에서 압도적 1위
• AI 시대의 시작을 알린 사건
